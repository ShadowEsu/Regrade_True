# Regrade 2.0 Design System

## Typography

- UI family: Inter/system sans fallback for controls, navigation, metadata, and body text.
- Editorial family: Regrade serif stack for hero statements and outcome moments only.
- Mobile scale: 11 metadata, 13 helper, 15 body, 17 row title, 22 section title, 30 page title, 38 editorial hero.
- Use weight before size to create hierarchy; avoid more than three weights on one screen.

## Spacing and shape

- Spacing unit: 4 px. Core rhythm: 8, 12, 16, 20, 24, 32, 40.
- Mobile content gutter: 18 px; compact card gap: 10–12 px.
- Radius: 10 controls, 14 rows, 18 cards, 24 feature cards, 28 sheets.
- Touch targets: minimum 44×44 px.

## Color roles

- Primary blue: navigation, progress, primary actions.
- Pale blue/lavender: selected surfaces and AI context.
- Warm yellow: needs-attention cards and today/streak emphasis.
- Mint: supported evidence, completed review, connected state.
- Soft red: failed connection, missing/unclear evidence, destructive action.
- Ink: near-black text. Canvas: white/pale gray.

## Components

Primary button, quiet button, destructive button, icon button, search field, segmented chips, status badge, metric tile, exam card, attention card, settings group/row, progress stepper, skeleton, toast, bottom sheet, notification row, connector card, annotation marker, evidence row, empty state, and inline Mr. Whale panel share tokens and motion primitives.
