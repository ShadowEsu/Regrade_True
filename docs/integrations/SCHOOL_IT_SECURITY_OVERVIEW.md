# School IT security overview

Regrade separates the static client, authenticated API, Firebase data, connector grants,
and AI processing. It uses least-privilege read scopes, encrypted connector credentials,
owner-scoped data, bounded uploads, rate limits, and explicit deletion. Appeals remain
student-controlled drafts.

Before a school pilot, provide architecture/data-flow diagrams, current subprocessors,
retention and incident procedures, test evidence for tenant isolation, a DPA as applicable,
and named security/privacy contacts. This document is an overview, not certification.
