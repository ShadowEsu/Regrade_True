# Tester onboarding

Use only synthetic or explicitly consented test papers. Create the assigned role, complete onboarding, skip connectors unless a staging connector is provided, and report the device/OS, exact steps, expected result, actual result, and screenshot with personal data removed.

Never send an appeal to a real educator during beta testing. Report privacy, cross-account, data-loss, billing, or account-access issues immediately and stop testing that flow.
