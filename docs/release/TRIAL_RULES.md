# Trial rules

- Product intent: a two-month Plus introductory trial.
- Eligibility must be determined by Apple/Google/RevenueCat, never a client boolean.
- The server must trust verified store entitlement state only.
- Trial start/end, renewal price, and cancellation terms must be visible before confirmation.
- Expired trials fall back to Free without deleting user work.

No live trial product was created or activated in this pass.
