# Support playbook

Priority 0: cross-user access, lost data, unauthorized submission, secret exposure, or billing against consent—disable the affected path and escalate immediately. Priority 1: auth/account deletion/purchase failures. Priority 2: connector, PDF, annotation, or AI workflow failures. Priority 3: cosmetic issues.

Collect app version, platform, timestamp, safe reproduction steps, and correlation id only. Never request passwords, tokens, full exam content, or store credentials.
