# Apple IAP setup

Create Free/Plus/Pro product mapping in App Store Connect, configure the eligible two-month Plus introductory offer, connect RevenueCat, add server webhook verification, and test purchase, restore, cancellation, billing retry, grace period, refund, and expiration in Sandbox/TestFlight.

Required founder inputs: paid-app agreement, tax/banking, product ids, App Store Connect API key, RevenueCat project, reviewed pricing text, and subscription legal links. No live product was changed.
