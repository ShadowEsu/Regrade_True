# Apple Store checklist

- [ ] Sign in with Apple configured wherever Google sign-in is offered
- [ ] RevenueCat App Store products and entitlements match server configuration
- [ ] Sandbox purchase, restore, renewal, expiry, refund, and family/account changes tested
- [ ] In-app account deletion works without contacting support
- [ ] Privacy manifest and App Privacy answers match actual collection
- [ ] Privacy, terms, support, and deletion URLs are public and stable
- [ ] Reviewer account/instructions use synthetic data only
- [ ] AI limitations, subscription terms, and user approval before sending are clear
- [ ] 1024px icon, screenshots, accessibility, and supported-device layouts verified
