# Supported content types

Accepted uploads: PDF, JPEG, PNG, HEIC/HEIF, and WebP within configured limits. The intended
content is previously marked exams, returned assignments, rubrics, score summaries, and
teacher feedback. The UI can render math/scientific notation and image-based handwritten work,
but difficult handwriting, diagrams, low-resolution scans, rotated pages, and incomplete
rubrics require uncertainty and user confirmation.
