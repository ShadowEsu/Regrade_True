# AI improvement plan

Regrade does not train a foundation model inside this repository. Improvement uses prompt
versioning, retrieval/configuration, structured extraction, human-reviewed test sets, and
provider model updates. Never use real student work for model improvement without a lawful
basis, explicit disclosure/consent where required, de-identification, and retention controls.

Maintain separate extraction, reasoning, tutoring, and appeal-drafting evaluations. Changes
ship only after regression, safety, uncertainty, handwriting, math, chemistry, and bias checks.
